== Informazioni ==

Se usi il software gestionale Slope e il sito della tua stuttura ricettiva è realizzato in WordPress, questo è il plugin che fa per te.

Aggiungi gli widget di Slope al sito web della tua struttura ricettiva! Questo plugin ti permette di posizionare dove vuoi la barra delle prenotazioni, i pacchetti e le promozioni, grazie agli shortcode.
Questo plugin è pensato per evitare ogni tipo di conflitto con il tema in uso sul tuo sito, quindi provalo senza pensieri!

== Installazione ==

1. Estrai la cartella del plugin all'interno della directory `/wp-content/plugins/`, oppure installalo direttamente dalla sezione plugin del tuo WordPress.
2. Attiva il plugin dall'apposita sezione.
3. Troverai la voce "Slope Widgets" sulla sidebar del tuo pannello di amministrazione. Configura le tue preferenze direttamente da lì.
4. Incolla lo shortcode **[slope-reservations]** nell'editor delle pagine, proprio dove vuoi che appaia il widget di prenotazione/promozione Slope. È facile!
